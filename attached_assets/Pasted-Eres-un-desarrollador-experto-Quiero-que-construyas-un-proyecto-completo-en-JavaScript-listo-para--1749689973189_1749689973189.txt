📦 Proyecto: mini-logger (Tiny Logging Library)
🎯 Descripción General:
Construye una pequeña librería de logging en JavaScript puro llamada mini-logger.
Debe ser paquetizable como NPM, usable en navegadores y Node.js.

El proyecto debe incluir:

La librería mini-logger como paquete independiente (estructura tipo src/mini-logger.js).

Una web minimalista de demostración donde pueda probar todos los features.

Documentación en la propia web con ejemplos de uso.

Instrucciones de instalación y uso en distintos frameworks.

🧩 Requerimientos de la Librería
1️⃣ Niveles de log soportados: debug, info, warn, error
2️⃣ Control de nivel de log: método setLevel(level)
3️⃣ Activar/desactivar logs: método enable(boolean)
4️⃣ Timestamp opcional: configurable
5️⃣ Contextos de log: método withContext('ModuleName')
6️⃣ Callback interceptores: método onLog(fn)
7️⃣ Colores en consola (solo navegador): usa console.log con estilo CSS para distinguir los niveles.
8️⃣ Zero dependencies: que sea 100% vanilla JS.

🔧 API de Ejemplo Esperada
javascript
Copy
Edit
import Logger from 'mini-logger';

const logger = new Logger({
  level: 'info',
  enabled: true,
  useTimestamp: true
});

logger.info("App started");
logger.debug("User data", user);
logger.error("API failed", error);

logger.withContext('Auth').warn("Invalid login");

logger.onLog((level, message, context) => {
  console.log("Intercepted:", level, message, context);
});
